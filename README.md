## Features

- Circular bubble mask reveal using `clip-path: circle()`
- Random bubble placement within defined zones
- Image switching with dots and prev/next navigation
- GSAP-powered scale animation with stagger
- Drop shadow on reveal

## Tech Stack

- **GSAP** (animations)
- **JavaScript (ES6+)**
- **SCSS**

## Reference

- [Segment Effect by Codrops](https://github.com/codrops/SegmentEffect/) — original concept and structure
